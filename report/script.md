# 🎙️ ROS 2 Humble 기반 TurtleBot3 블록 코딩 교육 플랫폼 - 발표 대본

---

### 📌 01. 도입 및 프로젝트 개요
* **대본:**
  안녕하세요. **'ROS 2 Humble 기반 TurtleBot3 블록 코딩 교육 플랫폼'** 발표를 맡은 BLOCK TEAM입니다.
  
  저희 프로젝트는 웹 기반 블록 코딩 환경과 ROS 2 기반 TurtleBot3를 연동하여, 초등학생도 쉽게 사용할 수 있는 교육용 로봇 제어 플랫폼을 구현하는 것을 목표로 하였습니다. 본 프로젝트는 총 5일(40시간) 동안 진행되었으며, Ubuntu 22.04 LTS와 ROS 2 Humble 버전을 개발 환경으로 사용하였습니다. 하드웨어로는 TurtleBot3 Burger를 활용하였고, 개발 언어로는 Python과 웹 표준 기술인 HTML5, CSS3, JavaScript를 사용했습니다.

### 📌 02. 프로젝트 소개 및 개발 목표
* **대본:**
  이번 프로젝트의 핵심 기능과 주요 목표에 대해 말씀드리겠습니다.
  
  먼저 사용자는 웹 브라우저에서 Drag & Drop 방식의 직관적인 인터페이스를 통해 초등학생도 쉽게 프로그램을 작성할 수 있습니다. 작성 후 실행 버튼을 누르면 블록 프로그램이 구조화된 데이터 형식인 **JSON Tree 형태로 변환**됩니다. 이 데이터는 **rosbridge_server를 통해 WebSocket 기반으로 ROS 2와 실시간 양방향 통신**을 하며, Interpreter Node가 이를 해석하여 로봇을 실시간으로 제어합니다. 
  
  개발 목표로는 직관적인 웹 UI 구축과 로봇 상태 동기화는 물론, **반복문과 조건문의 중첩 구조를 완벽히 지원하는 Stack 기반 파싱**, 그리고 LiDAR를 활용한 안전 주행 기능 구현을 설정하였습니다.

---

### 📌 03. 전체 시스템 아키텍처
* **대본:**
  다음은 시스템 구성입니다. 전체 시스템 아키텍처는 웹과 로봇 환경이 유기적으로 연결되어 있습니다.
  
  사용자가 웹 브라우저에서 블록 프로그램을 작성하면, rosbridge_server가 WebSocket을 통해 ROS 2로 전달합니다. 전달된 JSON Tree 데이터를 **Interpreter Node가 실시간으로 해석**하고, 로봇의 속도를 제어하는 `cmd_vel`, 현재 실행 중인 블록 상태를 전달하는 `run_state`, 그리고 부저를 제어하는 `buzzer` 토픽을 발행합니다. 이와 동시에 Ultrasonic Node가 LiDAR 데이터를 기반으로 장애물을 감지하며 실제 TurtleBot3 하드웨어를 안전하게 구동하게 됩니다.

---

### 📌 04. 주요 구현 기능 (블록 코딩, 인터프리터, 상태 동기화)
* **대본:**
  이어서 구체적인 구현 기능입니다. 
  
  웹 편집기에서는 프로그램의 실행과 정지뿐만 아니라 **현재 실행 중인 블록을 하이라이트** 해주는 직관적인 기능을 갖추었습니다. 또한, ROS 2 연동을 위해 `/program`, `/cmd_vel`, `/run_state` 토픽을 활용하여 안정적인 실시간 데이터 교환을 구현했습니다. 
  
  핵심 로직인 **JSON Tree 기반 Runtime Interpreter**는 블록 프로그램을 실시간으로 해석하여 제어 명령을 생성합니다. 특히 Loop나 If-Else 같은 반복문과 조건문이 겹쳐진 중첩 구조를 완벽히 처리하기 위해 **Stack 구조 기반의 런타임 파싱**을 도입하여 효율적인 메모리 관리와 복잡한 제어 흐름 처리가 가능하도록 했습니다.

---

### 📌 05. LiDAR 기반 안전 기능
* **대본:**
  교육용 플랫폼인 만큼 안전 주행 기능에도 만전을 기했습니다. 
  
  LiDAR 센서를 활용해 전방과 후방의 장애물을 실시간으로 동시에 탐지하고, 안전 거리를 기반으로 위험을 판단합니다. 만약 장애물이 감지되면 프로그램 전체를 강제 종료하는 것이 아니라, **현재 이동 중인 블록만 중단하고 다음 실행 흐름으로 자연스럽게 이어지도록 하는 'Runtime Skip' 구조**를 설계하여 흐름 끊김을 방지했습니다. 또한, 위험 상황 발생 시 즉시 정지하며 부저로 경고음을 발생하는 **'Emergency Stop'** 기능도 탑재하여 안전한 로봇 운용을 보장합니다.
